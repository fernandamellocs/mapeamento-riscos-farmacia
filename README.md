# Análise de riscos em operações em varejo farmacêutico
## Sobre este projeto
Este projeto apresenta uma análise de riscos baseada em situações comuns observadas em operações de varejo farmecêutico.
Os cenários descritos têm finalidade exclusivamente educacional e representam um estudo de Governança, Riscos e Compliance (GRC). Eles não descrevem ou avaliam uma empresa específica.
